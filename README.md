ClauseBuddy AI

A browser extension that helps users understand legal documents before they accept them.

ClauseBuddy scans Terms of Service, Privacy Policies, and similar legal pages and shows users what really matters in seconds. It highlights risky clauses, privacy concerns, and safety protections using a clear traffic-light system.

ğŸ§  How It Works

ClauseBuddy uses a hybrid system:

Local Scan (Always On)

Reads the full page in the browser

Detects key legal patterns

Shows real sentences from the website

Cloud AI (When Available)

Sends a small portion of the document to an AI model

Generates natural-language summaries and deeper explanations

If AI is unavailable, ClauseBuddy continues to work using the local scan.
